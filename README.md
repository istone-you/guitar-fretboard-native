# Guitar Fretboard - ギター学習アプリ

指板上に音名・度数・スケール・コードを重ね合わせて表示するギター学習ツール。

## 機能一覧

### 指板表示

- 6弦×15フレット（0〜14フレット）
- 音名表示 / 度数表示の切り替え
- ルート音の選択（全12音）
- フレット範囲の指定
- ダブルタップで横画面表示
- ポジションマーク（3, 5, 7, 9, 12フレット）

### レイヤーシステム

最大3つのレイヤーを重ねて表示。各レイヤーは独立した色を持ち、トグルで表示/非表示を切り替え可能。ドラッグで順序変更、複製ボタンで色違いコピー。同じセルに複数レイヤーが重なる場合は、同心円状に入れ子でドットが表示される（外側が1番目、内側が後のレイヤー）。

#### スケールレイヤー

対応スケール:
- 基本: メジャー、ナチュラルマイナー
- ペンタトニック: メジャーペンタ、マイナーペンタ
- ブルーノートスケール
- マイナー派生: ハーモニックマイナー、メロディックマイナー
- モード: Ionian, Dorian, Phrygian, Lydian, Mixolydian, Aeolian, Locrian
- 応用: フリジアンドミナント、リディアンドミナント、オルタード、ホールトーン、ディミニッシュ

#### コードレイヤー

表示形式:
- **コードフォーム**: 6弦/5弦ルートのバレーコード + オープンコードを枠付きで表示
  - 対応: Major, Minor, 7th, maj7, m7, m7(b5), dim7, m(maj7), sus2, sus4, 6, m6, dim, aug, 9, b9, #9, add9, 11, #11, add11, add#11, 13, b13
  - コード枠の表示/非表示切り替え
- **パワーコード**: ルート+5度の2音フォーム
- **トライアド**: 3弦セット（1-3弦, 2-4弦, 3-5弦, 4-6弦）×転回形（基本/第一/第二）
  - 対応: Major, Minor, Diminished, Augmented
- **ダイアトニック**: キー（メジャー/マイナー）×和音（3和音/4和音）から度数を選択
- **オンコード**: スラッシュコード（C/E, Am/G 等）のボイシングを表示
  - ルート音に応じた一覧から選択
  - 200以上のオンコードに対応

#### CAGEDレイヤー

- C・A・G・E・Dの5フォームを複数選択で表示
- メジャー / マイナーの切り替え
- コード枠の表示/非表示切り替え

#### カスタムレイヤー

- 音名または度数を自由に選択して指板上に表示
- 度数は拡張テンション対応（♭9, 9, ♯9, 11, ♯11, ♭13, 13）
- コードの構成音を抽出する機能（コードタイプを選択して抽出）
- リセット機能

**応用例:**
- コードタイプから構成音を抽出し、コードトーンの指板上の配置を把握
- 特定の度数だけを選んで、指板上のインターバルパターンを視覚化
- オクターブの位置関係を把握（同じ音名だけを選択して配置パターンを確認）
- スケールレイヤーと重ねて、コードトーンとスケール音の関係を視覚的に比較

#### プリセット

- 現在のレイヤー構成に名前を付けて保存
- 保存したプリセットをワンタップで読み込み
- プリセットの削除

### クイズ機能

5種類のクイズモード:
- **音名クイズ**: 指板上の音名を当てる（12択 / 指板タップ）
- **度数クイズ**: 指板上の度数を当てる（12択 / 指板タップ）
- **コード識別**: コードフォームからルート音とコード種別を回答
- **スケールクイズ**: スケールの構成音を選択
- **ダイアトニック**: ダイアトニックコードの構成を回答

### 設定

- テーマ: ダーク / ライト
- 臨時記号: ♯ / ♭
- 左利きモード: 指板を左右反転表示
- フレット範囲指定
- 言語: 日本語 / 英語

## 技術スタック

- **Runtime**: Bun
- **Framework**: Expo + React Native
- **Language**: TypeScript (strict mode)
- **UI**: React 19
- **SVG**: react-native-svg
- **i18n**: i18next + react-i18next
- **Storage**: @react-native-async-storage/async-storage

## 開発

```bash
# 依存のインストール
bun install

# 開発サーバー起動
bun start

# チェック
bun run fmt
bun run lint
bun run typecheck
bun run test
```
