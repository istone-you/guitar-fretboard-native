# Guitar Fretboard - ギター学習アプリ

指板上に音名・度数・スケール・コードを重ね合わせて表示するギター学習ツール。

## 機能一覧

### 指板表示

- 6弦×15フレット（0〜14フレット）
- 音名表示 / 度数表示の切り替え
- ルート音の選択（全12音）
- **レイヤーごとのルート**: ルートピッカーのシートでONにすると、各レイヤーに独立したルート音を設定可能。OFFのときは全レイヤーが共通ルートを使用
- フレット範囲の指定
- ダブルタップで横画面表示
- ポジションマーク（3, 5, 7, 9, 12フレット）

### レイヤーシステム

最大3つのレイヤーを重ねて表示。各レイヤーは独立した色を持ち、左のチェックボックスで表示/非表示を切り替え可能。iOS 26スタイルのインタラクションを採用:

- **タップ**: レイヤー行をタップして編集モーダルを開く
- **長押し**: iOS 26ネイティブ風コンテキストメニューを表示（編集・複製・削除）
- **左スワイプ**: 削除アクション
- **ドラッグ&ドロップ**: 行全体を長押し（200ms）してから縦にドラッグして並べ替え。シフト移動方式（間の要素も詰めて移動）。空スロットはドロップ先として有効だがドラッグ不可。`react-native-reanimated-dnd` を使用し、`layer.id` をキーとすることでドロップ後のちらつきを防止

同じセルに複数レイヤーが重なる場合は、同心円状に入れ子でドットが表示される（外側が1番目、内側が後のレイヤー）。

レイヤー設定モーダルの右上「?」ボタンから、レイヤータイプの解説と選択中のスケール・コードの解説を確認できる。レイヤータイプの説明は常に表示され、スケールやコードを切り替えると個別の解説のみが動的に切り替わる。

#### スケールレイヤー

対応スケール:
- 基本: メジャー、ナチュラルマイナー
- ペンタトニック: メジャーペンタ、マイナーペンタ
- ブルーノートスケール
- マイナー派生: ハーモニックマイナー、メロディックマイナー
- モード: Ionian, Dorian, Phrygian, Lydian, Mixolydian, Aeolian, Locrian
- 応用: フリジアンドミナント、リディアンドミナント、オルタード、ホールトーン、ディミニッシュ

#### コードレイヤー

表示形式:
- **コードフォーム**: 6弦/5弦ルートのバレーコード + オープンコードを枠付きで表示
  - 対応: Major, Minor, 5, dim, aug, 7th, maj7, m7, m7(b5), dim7, m(maj7), sus2, sus4, 7sus4, 6, m6, 9, b9, #9, add9, m(add9), 11, #11, add11, add#11, 13, b13
  - コード枠の表示/非表示切り替え
- **トライアド**: 3弦セット（1-3弦, 2-4弦, 3-5弦, 4-6弦）×転回形（基本/第一/第二）
  - 対応: Major, Minor, Diminished, Augmented
- **オンコード**: スラッシュコード（C/E, Am/G 等）のボイシングを表示
  - ルート音に応じた一覧から選択
  - 200以上のオンコードに対応

#### CAGEDレイヤー

- C・A・G・E・Dの5フォームを複数選択で表示
- メジャー / マイナーの切り替え
- コード枠の表示/非表示切り替え

#### カスタムレイヤー

- 音名または度数を自由に選択して指板上に表示
- 度数は拡張テンション対応（♭9, 9, ♯9, 11, ♯11, ♭13, 13）
- コードの構成音を抽出する機能（コードタイプを選択して抽出）
- リセット機能

**応用例:**
- コードタイプから構成音を抽出し、コードトーンの指板上の配置を把握
- 特定の度数だけを選んで、指板上のインターバルパターンを視覚化
- オクターブの位置関係を把握（同じ音名だけを選択して配置パターンを確認）
- スケールレイヤーと重ねて、コードトーンとスケール音の関係を視覚的に比較

#### プログレッションレイヤー

コード進行を"表示するため"ではなく、**他レイヤーを固定したままコードだけを順に差し替えるため**のレイヤー。

- **進行テンプレート**: I-IV-V / ii-V-I / I-vi-ii-V / I-V-vi-IV / i-VII-VI-VII / 12bar blues / Major-Triads / Major-7ths / Nat.Minor-Triads / Nat.Minor-7ths（固定）+ カスタム
- **コードサイズ**: 組み込みテンプレートはコードごとにタイプ固定。カスタムテンプレートはコードごとに個別指定
- **ステップ操作**: レイヤーパネルの ‹ / › ボタンで前後のコードへ即時移動
- **ゴースト表示**: 前後ステップのコードを同色の低透明度で重ねて表示（任意ON/OFF）
- 他レイヤー（スケール / CAGED / カスタム）と自由に組み合わせ可能

**活用例:**
- Progression + Scale: コード進行を辿りながら各コードでスケールの使われ方を比較
- Progression + CAGED: 進行の各コードをCAGEDフォームで把握
- Progression + Custom（3rd/7th）: ガイドトーンがどのコードでどこに現れるかを確認

#### プリセット

レイヤータブ内のプリセットボタンで管理。3ページ構成のモーダル:
- **一覧ページ**: 保存済みプリセットを一覧表示（名前・レイヤー数）。左スワイプで削除（赤背景＋ゴミ箱）。「+」ボタンで保存ページへ
- **保存ページ**: 名前を入力し、現在のレイヤー構成をプレビューして保存（レイヤーが0個の場合は無効）
- **詳細ページ**: 各レイヤーのタイプ・カラー・構成情報を読み取り専用で確認。Uploadボタンで現在のレイヤーをプリセット内容に置き換え

### Finder

親ページ（Selection）から各モードにスライドインで遷移する構成。Selection は ScrollView でスクロール可能。各モードの結果カードや一覧行は基本的にタップで詳細シート（BottomSheet）が開き、コードダイアグラム・解説・「レイヤーに追加」ボタンを表示する。

#### 構成音

指板をタップして音を選択し、その音の組み合わせに一致するコード/スケールを自動検出する機能。

- **入力方式**: 指板タップ ↔ 音名チップ選択を切り替え可能
- **ルート音設定**: 初期状態はルート未選択。指板を長押しでルート音を設定（追加音をクリア）
- **音の追加**: ルート設定後にタップで音を追加。タップした音名が全弦まとめてハイライト表示
- **リセット**: 選択チップのタップ、またはリセットボタンでクリア
- **完全一致**: 選択した音の集合がコードの構成音と完全に一致するものを表示（構成音数の多い順）
- **含まれる**: コードの構成音が全て選択音の中に収まるものを表示（構成音数の多い順）
- **含む**: 選択音が全てコードに含まれるもの（完成に近い順）を表示
- **構成音表示**: 音名/度数はレイヤータブのbaseLabelModeに連動
- **結果カードのアクション**: カードをタップするとBottomSheetが表示され、PillButtonで「レイヤーに追加」または「構成音をセット」を選択できる
  - 「レイヤーに追加」: レイヤータブへ遷移してレイヤーを追加（レイヤーが3つ埋まっている場合は無効）
  - 「構成音をセット」: Finder内の選択音を該当コード/スケールの構成音で更新

#### コードフォーム

コードタイプを一覧表示し、フォーム（コードダイアグラム）からコードを探す機能。

- **ルート音選択**: NotePickerButton でルート音を変更
- **コード一覧**: 全コードタイプをコードダイアグラム付きで表示。フォームが存在するコードのみ表示
- **詳細シート**: ダイアグラム・コード解説・「レイヤーに追加」PillButtonを表示

#### カポ

カポタスト位置とコード形を相互変換する機能。

- **実音を調べるモード**: カポフレット（0〜11）・押さえる形のキー・コードタイプを選択すると、実際に鳴るコード名とコードダイアグラムをカード内にインラインで表示。カードをタップするとBottomSheetが開き「レイヤーに追加」ボタンを表示
- **カポ位置を調べるモード**: 出したい音のキーと使うフォームのキーを選択すると、必要なカポフレット数を表示
- フレット数が9以上の場合は高フレット警告を表示

#### 五度圏

3リング円盤（外周=Major / 中周=Minor / 内周=m♭5）でキー同士の関係を視覚化するページ。Finder の選択画面からアクセスするほか、ダイアトニック / モーダルインターチェンジ / ドミナントモーションの各ページにある「五度圏で表示」ボタンからもサブページとして開ける（戻るで元のページへ戻る）。

- メジャーセグメントのタップでメジャーキー、マイナーセグメントのタップでマイナーキーを選択
- m♭5 セグメントのタップはキータイプ変更なしで位置のみ切替
- 中央に選択キー名と調号（♯/♭の数）を表示
- 表記は五度圏の慣例固定（右半=♯ / 左半=♭、6時位置は異名同音併記）
- 円盤下のトグルで重ねられるオーバーレイ（4種、同時1つのみ）:
  - **関連調**: 主調・属調・下属調・平行調を4色で枠表示
  - **ダイアトニック**: 選択キーの I–vii° を T/SD/D 機能色の枠で強調
  - **セカドミ+裏コード**: 各ダイアトニック音への V/X と ♭II/X を別色で外周に強調
  - **モーダルインターチェンジ**: 選択キーの平行調モードから借用できるコードを外周に強調
- 各オーバーレイの色凡例を円盤下に表示

#### ダイアトニック

キーのダイアトニックコードを一覧表示する機能。

- **ルート音選択**: NotePickerButton でキーのルート音を変更
- **キータイプ**: Major / Natural Minor を切り替え
- **コードサイズ**: 3和音 / 4和音 を切り替え
- **コード一覧**: 7つのダイアトニックコードを度数ラベル・**機能バッジ（トニック / サブドミナント / ドミナント）**・コードダイアグラム付きで表示
- **詳細シート**: ダイアグラム・機能解説・コード解説・「レイヤーに追加」PillButtonを表示

#### 関係調

キーを選んで関係のあるキー群（関係調・平行調・属調・下属調）を俯瞰する機能。

- NotePickerButton＋Major/Minorトグルで基準キーを選択
- 4種の関係調をカードで一覧表示
- カードタップ → ボトムシートで選択した関係調のダイアトニックコード一覧＋基準キーとのピボットコードを表示
- ピボットコードから「レイヤーに追加」が可能

#### ピボットコード

2つのキーを繋ぐ共通コード（ピボットコード）を検索する機能。

- 転調元キー（A）・転調先キー（B）をそれぞれ音名＋Major/Minorで選択
- 両キーのダイアトニックに共通するコードを一覧表示
- 各コードカードに「Key Aでの度数 / Key Bでの度数」を表示
- 「レイヤーに追加」ボタンでコードレイヤーとして追加可能

#### コードからキーを特定

複数のコードを入力してキーを推定する機能。

- **コード追加**: コードタイプを選択してリストに追加
- **キー候補表示**: 入力コードが含まれるキー（Major/Minor）を一致スコア順に表示

#### スケール互換

コードとスケールの互換関係を双方向で探索する機能。

**コード → スケール方向**

- **コード選択**: キー＋コードタイプの組み合わせで対象コードを複数追加
- **互換スケール一覧**: 全コードの構成音をカバーするスケールを、カテゴリー別カードでグループ表示
- **構成音ハイライト**: スケール行の各音名チップのうち、コードトーンを強調色で表示
- **詳細シート**: 指板プレビュー・構成音・「レイヤーに追加」ボタンを含むBottomSheetを表示

**スケール → コード方向**

- **スケール選択**: キー＋スケール種別（メジャー・マイナー・モード等）を選択
- **コードサイズ**: トライアド / 7thコードを切り替え
- **派生コード一覧**: スケール各音を根音とし、全音がスケール内に収まるコードを度数付きで表示
- **詳細シート**: コードダイアグラム・「レイヤーに追加」ボタンを含むBottomSheetを表示

#### モード

スケールとモードの親子関係を視覚化する機能。

- NotePickerButton＋モードピッカー（イオニアン〜ロクリアン、日本語名表示）で現在地を選択
- 親スケール（例: D ドリアン → C メジャー）をヘッダーに表示
- 7つのモードを番号付きで一覧表示
- モード行をタップすると詳細シートが開き、親スケール・解説テキスト・構成音チップ・「レイヤーに追加」ボタンを表示

#### 代理コード

コードの代理候補をタイプ別に表示する機能。

- **ルート音選択**: NotePickerButton でルート音を変更
- **コードタイプ**: Maj / m / maj7 / m7 / 7 の5種から選択
- **元コードカード**: 選択中のコードをカードで表示。タップで詳細シートが開く
- **ダイアトニック代理**: 同機能グループの代理コードを表示（VIm・IIIm 等、共通音を持つコード）
- **トライトーン代理**: ドミナント7th選択時のみ表示。半音6つ上の7thコード

#### テンション / アボイド

キーとコードから、コードトーン・テンション・アボイドを分類して表示する機能。

- **アボイド判定（b9）**: コードトーンの半音上にあるスケール音はアボイド
- **アボイド判定（トライトーン）**: コードの3度または♭3度と6半音（トライトーン）関係になるスケール音はアボイド
- **テンション表示**: アボイドに該当しないスケール音をテンションとして表示

#### モーダルインターチェンジ

平行調の各モードから借用できるコードを一覧表示する機能。

- **キー選択**: NotePickerButton＋Major/Minorトグルでキーを選択
- **借用コード一覧**: 借用元モード（エオリアン / ミクソリディアン / ドリアン / イオニアン / ハーモニックマイナー）ごとにセクション分けして、度数ラベル・コード名を縦リストで表示
- **「五度圏で表示」**: ボタンをタップするとモーダルインターチェンジオーバーレイ付きの五度圏サブページを開ける（戻るで元のページへ戻る）

#### ドミナントモーション

ドミナント解決のパターンを一覧・詳細で学べる機能。

- **キー選択**: NotePickerButton＋Major/Minorトグルでキーを選択
- **パターン一覧**: 以下6種のドミナントモーションパターンを行形式で表示
  - V→I（基本） / Ⅱ–Ⅴ–Ⅰ / 裏コード（♭II7→I）/ バックドア（♭VII7→I）/ ドミナント連鎖（V/V→V→I）/ ディミニッシュ解決
  - セカンダリードミナント（各ダイアトニックコードへの V/X → X）をセクションとして表示。セクションヘッダーの「五度圏で表示」ボタンからセカドミ+裏コードオーバーレイ付きの五度圏サブページを開ける
- **詳細シート**: 行をタップすると以下を表示
  - ドミナントコードと解決先コードのダイアグラム（矢印で連結）
  - パターンの音楽的解説テキスト
  - ボイスリーディング：3rd・7th の音名・移動量（半音/全音/増4度）・方向（↑↓）
  - 「コード進行分析で確認」ボタンでサブページとして ProgressionAnalyzer を開く（戻るで元のページへ戻る）

#### コード進行分析

入力したコード進行から、最も可能性の高いキーと各コードの機能を分析する機能。

- **入力モード**: 音名入力（キー＋コードタイプを選択）/ 度数入力（キーを固定してダイアトニック度数で入力）を切り替え可能
- **コード追加**: 指板をタップして音名を選び、コードタイプ（Major / Minor / 7th 等）を指定して進行に追加
- **テンプレート読み込み**: 既存のコード進行テンプレートを読み込んで分析対象に設定
- **キー分析結果**: 入力コードとの一致スコアが高い順に最大5つのキー候補（Major / Minor）を表示
- **コード機能表示**: 各候補キーにおける各コードの機能（T / SD / D）・度数・借用和音・セカンダリードミナントをラベル表示

#### コードサジェスト

単体のコードを選択し、次に進められるコード候補をカテゴリ別に一覧表示する機能。

- **コード入力**: ルート音（NotePickerButton）＋コードタイプチップ（Major / Minor / maj7 / m7 / 7th）で入力コードを選択
- **カテゴリ別候補**: 以下8カテゴリで候補を整理して表示
  - ダイアトニック遷移・Ⅱ–Ⅴの入口・セカンダリードミナント候補・裏コード候補
  - 終止方向・バックドア候補・経過和音候補・借用和音候補
- **詳細シート**: コードダイアグラムと度数ラベルを表示し、「レイヤーに追加」が可能

### コード進行テンプレート

ユーザー作成のコード進行テンプレートを管理する。

- ＋ボタン → 作成シート（名前入力・度数セレクター＋コードタイプ選択でコードを追加、Triad/Seventh/Tensionグループから選択、混在可）
- テンプレート行をタップ → 詳細ペインが右からスライドイン
  - テンプレート名・キー選択（NotePickerButton でトランスポーズ）
  - 各コードのコードダイアグラム（全ボイシング）を度数ごとに表示
  - Uploadボタン: コード進行レイヤーとしてレイヤータブに追加（レイヤーが3つ埋まっている場合は無効）
  - ⋯ボタン: 編集シートを開く
  - 右スワイプで詳細ペインを閉じる
- 左スワイプで削除（赤背景＋ゴミ箱）
- ドラッグハンドルを長押し→ドラッグで並べ替え
- 作成したテンプレートはレイヤー設定の「コード進行」テンプレートとして利用可能

### クイズ機能

5種類のクイズモード:
- **音名クイズ**: 指板上の音名を当てる（12択 / 指板タップ）
- **度数クイズ**: 指板上の度数を当てる（12択 / 指板タップ）
- **コード識別**: コードフォームからルート音とコード種別を回答
- **スケールクイズ**: スケールの構成音を選択
- **ダイアトニック**: ダイアトニックコードの構成を回答

#### 苦手分析

クイズ画面内から開ける、回答履歴に基づいて苦手傾向を可視化するページ。

- **弦×フレット ヒートマップ**: 指板レイアウトと同じ向きで正答率を色で表示（赤=苦手 / 緑=得意）
- **モード別**: 音名・度数・コード・スケール・ダイアトニックの各クイズモードごとの正答率
- **音名別**: 全12音の正答率ランキング
- **度数別**: 全12度数の正答率ランキング
- **弦別**: 6弦それぞれの正答率ランキング
- **フレット別**: 0〜14フレットそれぞれの正答率ランキング
- **コード種別**: 全コードタイプの正答率ランキング
- **スケール別**: 全スケールタイプの正答率ランキング
- セクションは折りたたみ式、ヒートマップのみ常時展開
- データが5問未満のエントリは薄く表示（`--`）、0問は未解答として区別
- 記録はアプリを閉じても保持（AsyncStorage、最大1000件）
- リセットボタンで全記録を削除

### 設定

- テーマ: ダーク / ライト
- 臨時記号: ♯ / ♭
- 左利きモード: 指板を左右反転表示
- フレット範囲指定
- 言語: 日本語 / 英語

## UI デザイン（iOS 26）

iOS 26 のデザインパターンに準拠したUIを採用。

- **ヘッダータイトル**: 各タブのページタイトルを左大文字で表示（レイヤー / Finder / テンプレート / クイズ）
- **ルート音・表示切替**: ヘッダーから指板下へ移動。指板との関連性を視覚的に明確化
- **同心円角丸**: ボトムシート（28px）と内部要素の角丸を同心円ルールで統一
- **クイズスワイプ**: クイズパネルの左端に `borderTopLeftRadius: 28 / borderBottomLeftRadius: 28` を付与
- **GlassIconButton**: ×・‹・? などアイコン系ボタンを `expo-glass-effect` の `GlassView`（iOS Liquid Glass）で統一した共通コンポーネント（44pt 標準サイズ）
- **ハンドル余白**: 共通定数 `SHEET_HANDLE_CLEARANCE = 28` を `BottomSheetModal` からエクスポートし、全ボトムシートのコンテンツ上部余白を統一
- **Progressive Blur ヘッダー**: `SheetProgressiveHeader` 共通コンポーネントで iOS 26 風のフロストガラスヘッダーを全ボトムシートに適用。`expo-blur` の `BlurView` が全幅を覆い、`expo-linear-gradient` の `LinearGradient` がシート背景色へグラデーションでフェードする
